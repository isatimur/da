### How to Use

**Adding Tasks:**
1. Click the task manager icon (T) or press 'T' key
2. Click "Add Task"
3. Fill in title, description, priority, category, due date, and tags
4. Save your task

**Today's Focus Widget:**
- Automatically shows your top 5 priority tasks
- Complete tasks directly from the widget
- Click "View All Tasks" to open full manager

**Using the Pomodoro Timer:**
1. Open Task Manager
2. Select "Pomodoro" tab
3. Choose a task to focus on
4. Set your desired duration
5. Click "Start"

**Keyboard Shortcuts:**
- Press 'T' - Open Task Manager
- Press 'Space' - Get new affirmation
- Press 'B' - Start breathing exercise
- Press 'S' - Open settings
- Press 'F' - Toggle focus mode
- Press 'M' - Open menu
- Press 'C' - Copy affirmation
- Press 'Esc' - Close dialogs
